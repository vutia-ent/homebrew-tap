# Vutia Enterprise Homebrew Tap

Homebrew formulae for Vutia Enterprise projects.

## Installation

```bash
brew tap vutia-ent/tap
```

## Available Formulae

### Fastpy CLI

Create production-ready FastAPI projects with one command.

```bash
brew install fastpy
```

#### Usage

```bash
# Create a new project
fastpy new my-api

# Show version
fastpy version

# Open documentation
fastpy docs
```
